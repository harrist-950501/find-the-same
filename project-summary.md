## Overview of Project

Find The Same is a single-page browser memory game that uses JavaScript to generate a randomized set of matching card pairs, handle card-flip interactions, track elapsed time, update the match counter, and display start and end dialogs during gameplay. The project combines structured HTML, custom CSS styling, and DOM-driven interactivity to create a complete front-end game experience.
