# Find The Same

A browser-based memory matching game built with HTML, CSS, and vanilla JavaScript.

## Overview

Find The Same is a single-page web game where players flip cards to uncover matching pairs of animal icons. The interface includes a start dialog, live timer, match counter, reset control, dynamically generated game board, and a completion dialog shown when all pairs have been matched.

## Features

- Randomized card placement generated with JavaScript
- Click-based card flipping and match detection
- Live timer that tracks elapsed play time
- Match counter that updates as pairs are found
- Reset flow for starting a new round
- Start and end dialogs to guide gameplay

## Built With

- HTML5
- CSS3
- JavaScript
- Google Fonts

## Purpose

This project demonstrates core front-end development skills through an interactive game interface, including DOM manipulation, event handling, dynamic content generation, and UI state updates in the browser.
